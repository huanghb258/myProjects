全自定义导航
	说明：主要写给自己用+练手，不考虑兼容性，目前还有很多功能还没写好
	对我自己来说可完全替代导航网+收藏夹/书签栏+云笔记+思维导等，如果想，甚至可以代替桌面
	特色功能：
	1.跟大多数导航一样有个百度搜索，支持搜索提示+历史搜索,不同点在于切换网页、图片、地图搜索时搜索提示
	也会跟着变，包括历史搜索记录

	2.自定义常用网址，增加、删除、调整位置顺序随心所欲，使用localStorage存储，可导出到JSON(还没弄)

	3.最主要的点，支持读取本地文件生成列表到左侧，可以右键直接打开txt文件或目录，也能直接渲染到页面
	
	平时喜欢用文本文档作笔记，写多了自己都找不到。等写好了文档内容搜索功能，就能完美解决了


使用：
	运行环境：
		装个node就行了，如果可以再装个pm2更好:npm install pm2 -g
		注：只在win7和win10上测试使用过

	目录结构：
		nav-server整个文件夹放在myProjects文件夹下，什么盘不管，结构不能改

	然后cd到nav-server目录下：
		npm install
		npm start
	如果装了pm2：
		npm install
		pm2 start pm2.json

备注：
	遍历文件时以下文件夹会被排除：
		文件夹名称为 img、js、css以及以点开头的文件名
		且只加载.txt和.html文件
		
	在win7下使用打开文件或目录功能cmd窗口会闪一下
