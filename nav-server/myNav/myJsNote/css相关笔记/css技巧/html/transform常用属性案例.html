注意transform-origin并不是transform中的属性值,以下属性基本上都可以配合使用

它的作用是改变元素基点（如配合旋转时改变旋转基点）,值可以是bottom、top、center...

使用示例：transform-origin:(left,top) //设置基点放左上角

transform-origin:right //基点为右边中间点 transform-origin(25%,75%) //...
