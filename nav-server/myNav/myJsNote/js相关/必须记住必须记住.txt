数字转换：

	parseInt:不管小数，只取整数，支持数字加字符串
		parseInt(1.9)//1	parseInt('1a')//1	parseInt('a1');
NaN

		注意：据说老版本parseInt('08'); // 0  

	toFixed(保留位数):保留指定小数位数，少则加零，多则四舍五入，如下
		例：5.6.toFixed(2)  //5.60	5.456.toFixed(2)   //5.46

		注意：只接受小数（整数或字符串等会报错），且返回的是字符串
