
手动创建：
	1.进入分支仓库    //huanghb258/demo

	2.点击Branch创建gh-pages        //名称不能变

	3.点击settings --> 选Branchs

	4.两个下拉菜单都选gh-pages  Update也点一下

	5.点完会自动跳转保存页，点保存即可

	访问：huanghb258.github.io/demo       //设置完可能要等几分钟



/**********************************/

命令创建：
	查看分支状态：
		git branch

	创建分支：
		git checkout -b gh-pages	//创建gh-pages分支

	删除分支：
		git push origin --delete 分支名

	切换分支：
		git checkout 分支名
