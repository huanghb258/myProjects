说明：
	下面的"."代表所有文件(当前目录)，如果只想设置单个文件，把点改成文件名就好了
	git bash下双击tap可以代码提示补全


git status .：
	查看当前路径下的的状态。如果两个点代表上一个目录

git add .: 
	把工作区的所有变化，都添加到 版本库/暂存区。

git commit -m "提交时说明": 
	对本次变化作一个简单的说明。

git push: 
	把暂存区的所有更新内容， 都推送到远程服务器。

git pull: 
	把代码从远程服务器同步到本地。

git log：
	查看提交历史

git diff:
	比较工作区与暂存区的差异，就是看看你到底都做了什么修改。

git clone url地址: 
	将远程项目克隆到本地（第一次拉项目时使用,后面的更新都用git pull）
